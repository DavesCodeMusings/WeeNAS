About

The play on the FreeNAS name is intentional. FreeNAS is an incredibly robust open-source network-attached storage software distribution based on FreeBSD and OpenZFS. It also has correspondingly robust system requirements, and that in turn drives up the cost of ownership.

WeeNAS, on the other hand, eschews all that fancy, high-performance hardware and ZFS filesystem. WeeNAS is designed to run on a cheap Raspberry Pi 2 and uses a UFS filesystem on a common USB flash drive for storage.

Comparison to FreeNAS

  • Is WeeNAS faster? No.
  • Is WeeNAS more fault tollerant? No.
  • Should you entrust your most precious files to WeeNAS? No.

So what is WeeNAS good for?

WeeNAS is basically a network attached flash drive. Use it like a flash drive. Plus, it's fun to say.

Personally, I created WeeNAS as a way to keep a second copy of the documents and other files I have on my PC. The hardware is so inexpensive, I'm seriously considering building another, identical system with a scheduled rsync between them.

In other words, I'm banking on having multiple copies of my important files in case something goes sideways, rather than a one copy on a single super-robust system.

All of this and more!

WeeNAS is a do-it-yourself project lover's dream. You supply the hardware, and WeeNAS guides you through the rest.

WeeNAS will provide you with:

  • Customizable, automated install script to get you started quickly (work in progress)
  • REST API for easy management of the system without needing the command-line (planning phase)
  • Web-based administration system for day to day tasks (planning phase)

System requirements

Here's what you'll need for hardware:

  • Raspberry Pi 2 Model B and suitable power supply
  • 32G micro-sd card compatible with Raspberry Pi
  • A USB flash drive compatible with Raspberry Pi *
  • An Ethernet cable and home router with at least one free RJ-45 LAN connection
  • Optional, externally-powered spinning disk for backups
  • A client computer capable of attaching to SMB (CIFS) shares

* The USB flash drive is where your home directories are stored. Get the largest capacity that fits your budget.

Client Software

Here's what you'll need for software:

  • 7-Zip or similar tool to decompress the FreeBSD image file
  • Win32DiskImager or similar software to write the image file to the SD card
  • AngryIP Scanner to help you find the DHCP address of your Raspberry Pi
  • PuTTY or other SSH client to log in for the initial configuration
