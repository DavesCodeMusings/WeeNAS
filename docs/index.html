About

The play on the FreeNAS name is intentional. FreeNAS is an incredibly robust open-source network-attached storage software distribution based on FreeBSD and OpenZFS. It also has correspondingly robust system requirements, and that in turn drives up the cost of ownership.

WeeNAS, on the other hand, eschews all that fancy, high-performance hardware and sets its sites on the inexpensive Raspberry Pi with a UFS filesystem and common USB flash drive for storage.

Comparison to FreeNAS

  • Is WeeNAS faster? No.
  • Is WeeNAS more fault tollerant? No.
  • Should you entrust your most precious files to WeeNAS? No.

So what is WeeNAS good for?

WeeNAS is basically a network attached flash drive. Use it like a flash drive.

Personally, I created WeeNAS as a way to keep a second copy of the documents and other files I have on my PC. The hardware is so inexpensive, I'm seriously considering building another, identical system with a scheduled rsync between them.

In other words, I'm banking on having multiple copies of my important files in case something goes sideways, rather than a one copy on a single super-robust system.

DIY bliss and more!

WeeNAS is a do-it-yourself project lover's dream. You supply the hardware, and WeeNAS guides you through the rest.

WeeNAS is in its infancy now, but soon it will provide you with:

  • Easy to follow documentation to guide you on your journey. (Work in progress)
  • Customizable, automated install script to get you started quickly. (Finished but untested)
  • REST API for easy management of the system without needing the command-line. (Planning phase)
  • Web-based administration page leveraing REST API for day to day tasks. (Planning phase)
