Prerequisites

System Hardware

  • Raspberry Pi 2 Model B and suitable power supply
  • 32G micro-sd card compatible with Raspberry Pi
  • A USB flash drive compatible with Raspberry Pi *
  • Optional, externally-powered spinning disk for backups
  • An Ethernet cable and home router with at least one free RJ-45 LAN connection
  • A client computer capable of attaching to SMB (CIFS) shares

* The USB flash drive is where your home directories are stored. Get the largest capacity that fits your budget.

System Software

  • FreeBSD 12.1 for Raspberry Pi 2

Windows Client Software

  • 7-Zip or similar tool to decompress the FreeBSD image file
  • Win32DiskImager or similar software to write the image file to the SD card
  • AngryIP Scanner to help you find the DHCP address of your Raspberry Pi
  • PuTTY or other SSH client to log in for the initial configuration

Preparing the SD Card Image

If you are familiar with the Raspberry Pi, the procedure for installing FreeBSD onto the SD Card is no different than installing Raspbian or any of the other Raspberry Pi distributions.

  1. If you downloaded an XZ compressed image from the FreeBSD site, extract it with 7-zip by right-clicking and selecting 7-zip > Extract Here from the context menu.
  2. Start Win32DiskImager. Select the FreeBSD image file and the drive letter of your Micro SD card. Write the image.
  3. Eject the Micro SD card from the PC and place it in the Raspberry Pi.

Connecting the Raspberry Pi

  1. Use an Ethernet cable to plug your Raspberry Pi into your home router. (There is no wireless option for WeeNAS.)
  2. Do not plug in any USB devices like the flash drive or backup disk yet.
  3. Plug the Raspberry Pi into its power supply and wait patiently as it boots up. (First boot can take a while.)
  4. When the red LED goes off and the Ethernet link light begins flashing, FreeBSD is booted up and ready.
